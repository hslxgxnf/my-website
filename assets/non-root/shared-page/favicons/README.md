# Images

### "\_" in a favicon image name means this image is for a special case.

### Except for special cases, a favicon image name should keep its site name as much as possible.

### "-" in a favicon image name means a space.

### Examples

- GitHub → GitHub
- Next.js → Next.js
- Stack Overflow → Stack-Overflow

# Variables

### A variable name should keep its site name as much as possible.

### "$" in a variable name means a dot.

### "\_" in a variable name means a space.

### Examples

- GitHub → GitHub
- Next.js → Next$js
- Stack Overflow → Stack_Overflow
